# How to use Components

## UserProfileCard
The user-profile-card component requires passing a user object which holds string values for  a user's  name, email, role, and avatarUrl. The user's id is a number and the user's email and role is optional. 

## ProductDisplay

The product-display component requires passing a product object which holds  string values for  a product's  name, price(number value), showDescription(boolean), imageURL, inStock(boolean) and id thats a number value. The description, image, and stock-status are optional.
